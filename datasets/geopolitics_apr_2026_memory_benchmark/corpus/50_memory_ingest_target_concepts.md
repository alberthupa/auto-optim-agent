# Target concepts a good memory-ingest system should preserve

This corpus is intentionally shaped around reusable concepts rather than isolated facts alone. A strong ingest system should preserve concept nodes such as geoeconomic confrontation, strategic chokepoints, sanctions, burden sharing, gray-zone pressure, critical minerals, energy security, and tactical dialogue.

## Key points

- Harder benchmark questions rely on these shared concepts more than on single-document summaries.
- Entity notes and timeline notes still matter, but they are not sufficient by themselves.
- The best vault structure will combine source-specific notes with cross-cutting concept notes.

## Why this matters for the benchmark

This file gives the benchmark an explicit theory of what good memory structure should look like.
