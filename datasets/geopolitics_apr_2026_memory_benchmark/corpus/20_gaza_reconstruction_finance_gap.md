# Gaza reconstruction and the finance gap

The reconstruction effort in Gaza is hampered by the difference between pledges and disbursements. In the corpus, that gap blocks governance plans and demonstrates how political declarations can fail at the implementation stage.

## Key points

- Large sums were pledged, but only a small portion had arrived.
- The shortage created operational paralysis.
- The funding gap weakened the credibility of the broader reconstruction push.

## Why this matters for the benchmark

This file is useful for testing whether the memory system tracks the difference between promised resources and available resources.
